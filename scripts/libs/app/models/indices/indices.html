Nasdaq Futures Overview

 
This page contains data on the E-mini Nasdaq 100 Futures CFDs. The NASDAQ-100 Index is a modified capitalization-weighted index of the 100 largest and most active non-financial domestic and international companies listed on the NASDAQ. The fact that index does not include financial companies as those were placed in a different indices; as does the fact that the index is not compiled of purely United States based companies - differentiates and distinguishes it from similar indices. More information can be found in other sections of this page, via the navigation tabs; real time CFD data, historical data, charts, technical analysis and a user discussion area.
